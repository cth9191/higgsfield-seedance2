# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| 1.0.x | ✅ Yes |

## Reporting a Vulnerability

This repository contains Claude skill files (Markdown text) — no executable code or API keys. Security risk is low, but if you find any issues, please report via:

1. **GitHub Issues** — For non-sensitive issues
2. **𝕏 DM @higgsfield** — For sensitive issues

### What We Care About

- Accidentally included API keys or credentials in skill files
- Prompt injection risks (skill content that could be misused)
- Compromised third-party links

### Response Time

- Acknowledgment: within 48 hours
- Fix plan: within 7 days
